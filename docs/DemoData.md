Demo Data
==========

Two role-specific fixture bundles live in the iOS target and can be loaded directly on-device for demos:

- `ios/DJMediWallet/DJMediWalletXCode/DJMediWallet25/DJMediWallet25/Resources/TestData/PatientRecords.json`
- `ios/DJMediWallet/DJMediWalletXCode/DJMediWallet25/DJMediWallet25/Resources/TestData/PractitionerTasks.json`

From the Settings → About screen, tap the About icon five times in quick succession to import the bundle that matches the current profile role.

Switching between Patient ↔ Practitioner roles automatically purges any previously loaded demo credentials so that the wallet reflects the active role’s dataset.
